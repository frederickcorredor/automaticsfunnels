Despachador Automatizado para Sillas de Ruedas

El sistema automatizado para el control de préstamo de sillas de ruedas desarrollado por GIGAWATT, facilita el proceso tanto al cliente como al centro comercial, reduciendo el tiempo necesario para la solicitud de préstamo, mientras libera el tiempo de la persona encargada por parte de la plataforma comercial en el registro de la silla de ruedas.

Este sistema facilita toda la operación de préstamo, manteniendo las sillas ancladas a un seguro mecánico, que únicamente se libera cuando un usuario necesita utilizar una silla, proceso que se realiza en menos de 2 minutos y sólo requiere de la cédula de ciudadanía del usuario, para devolver la silla, basta con ubicarla en el mismo lugar del cual se tomó.

El software instalado permite obtener reportes de uso del sistema para la gestión de activos, como la cantidad de préstamos de cada silla, datos de los usuarios con los cuales se pueden hacer estadísticas de uso, y demás procesos administrativos.
